Vue3 е“Қеә”ејҸеҺҹзҗҶ

е®һзҺ°еҺҹзҗҶ:

  • йҖҡиҝҮ ProxyпјҲд»ЈзҗҶпјү: жӢҰжҲӘеҜ№иұЎдёӯд»»ж„ҸеұһжҖ§зҡ„еҸҳеҢ–, еҢ…жӢ¬пјҡеұһжҖ§еҖјзҡ„иҜ»еҶҷгҖҒеұһжҖ§зҡ„ж·»еҠ гҖҒеұһжҖ§зҡ„еҲ йҷӨзӯүгҖӮ
  • йҖҡиҝҮ ReflectпјҲеҸҚе°„пјү: еҜ№жәҗеҜ№иұЎзҡ„еұһжҖ§иҝӣиЎҢж“ҚдҪңгҖӮ
  • MDN ж–ҮжЎЈдёӯжҸҸиҝ°зҡ„ Proxy дёҺ Reflectпјҡ
    • Proxyпјҡhttps://developer.mozilla.org/zh-CN/docs/Web/JavaScript/Reference/Global_Objects/Proxy
    • Reflectпјҡhttps://developer.mozilla.org/zh-CN/docs/Web/JavaScript/Reference/Global_Objects/Reflect

Vue2 зҡ„е“Қеә”ејҸ

  • е®һзҺ°еҺҹзҗҶпјҡ

    • еҜ№иұЎзұ»еһӢпјҡйҖҡиҝҮObject.defineProperty()еҜ№еұһжҖ§зҡ„иҜ»еҸ–гҖҒдҝ®ж”№иҝӣиЎҢжӢҰжҲӘпјҲж•°жҚ®еҠ«жҢҒпјүгҖӮ

    • ж•°з»„зұ»еһӢпјҡйҖҡиҝҮйҮҚеҶҷжӣҙж–°ж•°з»„зҡ„дёҖзі»еҲ—ж–№жі•жқҘе®һзҺ°жӢҰжҲӘгҖӮпјҲеҜ№ж•°з»„зҡ„еҸҳжӣҙж–№жі•иҝӣиЎҢдәҶеҢ…иЈ№пјүгҖӮ

  • еӯҳеңЁй—®йўҳпјҡ

    • ж–°еўһеұһжҖ§гҖҒеҲ йҷӨеұһжҖ§, з•ҢйқўдёҚдјҡжӣҙж–°гҖӮ
    • зӣҙжҺҘйҖҡиҝҮдёӢж Үдҝ®ж”№ж•°з»„, з•ҢйқўдёҚдјҡиҮӘеҠЁжӣҙж–°гҖӮ

reactive еҜ№жҜ” ref

  • д»Һе®ҡд№үж•°жҚ®и§’еәҰеҜ№жҜ”пјҡ
    • ref з”ЁжқҘе®ҡд№үпјҡеҹәжң¬зұ»еһӢж•°жҚ®гҖӮ
    • reactive з”ЁжқҘе®ҡд№үпјҡеҜ№иұЎпјҲжҲ–ж•°з»„пјүзұ»еһӢж•°жҚ®гҖӮ
    • еӨҮжіЁпјҡref д№ҹеҸҜд»Ҙз”ЁжқҘе®ҡд№үеҜ№иұЎпјҲжҲ–ж•°з»„пјүзұ»еһӢж•°жҚ®, е®ғеҶ…йғЁдјҡиҮӘеҠЁйҖҡиҝҮreactiveиҪ¬дёәд»ЈзҗҶеҜ№иұЎгҖӮ
  • д»ҺеҺҹзҗҶи§’еәҰеҜ№жҜ”пјҡ
    • ref йҖҡиҝҮObject.defineProperty()зҡ„getдёҺsetжқҘе®һзҺ°е“Қеә”ејҸпјҲж•°жҚ®еҠ«жҢҒпјүгҖӮ
    • reactive йҖҡиҝҮдҪҝз”ЁProxyжқҘе®һзҺ°е“Қеә”ејҸпјҲж•°жҚ®еҠ«жҢҒпјү, е№¶йҖҡиҝҮReflectж“ҚдҪңжәҗеҜ№иұЎеҶ…йғЁзҡ„ж•°жҚ®гҖӮ
  • д»ҺдҪҝз”Ёи§’еәҰеҜ№жҜ”пјҡ
    • ref е®ҡд№үзҡ„ж•°жҚ®пјҡж“ҚдҪңж•°жҚ®йңҖиҰҒ.valueпјҢиҜ»еҸ–ж•°жҚ®ж—¶жЁЎжқҝдёӯзӣҙжҺҘиҜ»еҸ–дёҚйңҖиҰҒ.valueгҖӮ
    • reactive е®ҡд№үзҡ„ж•°жҚ®пјҡж“ҚдҪңж•°жҚ®дёҺиҜ»еҸ–ж•°жҚ®пјҡеқҮдёҚйңҖиҰҒ.valueгҖӮ

дёӘдәәе»әи®®зҡ„еҶҷжі•жҳҜиҝҷж ·пјҢжңҖеҗҺ return д»–зҡ„ data

setup зҡ„дёӨдёӘжіЁж„ҸзӮ№

  • setup жү§иЎҢзҡ„ж—¶жңә
    • еңЁ beforeCreate д№ӢеүҚжү§иЎҢдёҖж¬ЎпјҢthis жҳҜ undefinedгҖӮ
  • setup зҡ„еҸӮж•°
    • propsпјҡеҖјдёәеҜ№иұЎпјҢеҢ…еҗ«пјҡз»„д»¶еӨ–йғЁдј йҖ’иҝҮжқҘпјҢдё”з»„д»¶еҶ…йғЁеЈ°жҳҺжҺҘж”¶дәҶзҡ„еұһжҖ§гҖӮ
    • contextпјҡдёҠдёӢж–ҮеҜ№иұЎ
      • attrs: еҖјдёәеҜ№иұЎпјҢеҢ…еҗ«пјҡз»„д»¶еӨ–йғЁдј йҖ’иҝҮжқҘпјҢдҪҶжІЎжңүеңЁ props й…ҚзҪ®дёӯеЈ°жҳҺзҡ„еұһжҖ§, зӣёеҪ“дәҺ this.$attrsгҖӮ
      • slots: ж”¶еҲ°зҡ„жҸ’ж§ҪеҶ…е®№, зӣёеҪ“дәҺ this.$slotsгҖӮ

и®Ўз®—еұһжҖ§дёҺзӣ‘и§Ҷ

1.computed еҮҪж•°

  • дёҺ Vue2.x дёӯ computed й…ҚзҪ®еҠҹиғҪдёҖиҮҙ

  • еҶҷжі•пјҢдҪҶжҳҜ name1 е’Ң name2 дёӯй—ҙзҡ„з©әж јеҲ жҺүзҡ„иҜқдјҡеҮә bug

2.watch еҮҪж•°

  • дёҺ Vue2.x дёӯ watch й…ҚзҪ®еҠҹиғҪдёҖиҮҙ

  • дёӨдёӘе°ҸвҖңеқ‘вҖқпјҡ

    • зӣ‘и§Ҷ reactive е®ҡд№үзҡ„е“Қеә”ејҸж•°жҚ®ж—¶пјҡoldValue ж— жі•жӯЈзЎ®иҺ·еҸ–гҖҒејәеҲ¶ејҖеҗҜдәҶж·ұеәҰзӣ‘и§ҶпјҲdeep й…ҚзҪ®еӨұж•ҲпјүгҖӮ
    • зӣ‘и§Ҷ reactive е®ҡд№үзҡ„е“Қеә”ејҸж•°жҚ®дёӯжҹҗдёӘеұһжҖ§ж—¶пјҡdeep й…ҚзҪ®жңүж•ҲгҖӮ

  • жҷ®йҖҡ ref еұһжҖ§зӣ‘и§Ҷзҡ„ж—¶еҖҷдёҚйңҖиҰҒеҠ .valueпјҢзӣ‘и§Ҷзҡ„жҳҜеұһжҖ§пјҢдёҚжҳҜе…¶дёӯзҡ„еҖј

  • дҪҶжҳҜеҜ№иұЎзұ»еһӢзҡ„жң¬еә”иҜҘз”Ё reactive зҡ„з”ЁдәҶ refпјҢе°ұйңҖиҰҒ.valueпјҢеӣ дёәд»–зҡ„ value йҮҢйқўжүҚжҳҜ ProxyпјӣжҲ–иҖ…зӣҙжҺҘејҖеҗҜ deep

3. watchEffect еҮҪж•°

  • watch зҡ„еҘ—и·ҜжҳҜпјҡж—ўиҰҒжҢҮжҳҺзӣ‘и§Ҷзҡ„еұһжҖ§пјҢд№ҹиҰҒжҢҮжҳҺзӣ‘и§Ҷзҡ„еӣһи°ғгҖӮ

  • watchEffect зҡ„еҘ—и·ҜжҳҜпјҡдёҚз”ЁжҢҮжҳҺзӣ‘и§Ҷе“ӘдёӘеұһжҖ§пјҢзӣ‘и§Ҷзҡ„еӣһи°ғдёӯз”ЁеҲ°е“ӘдёӘеұһжҖ§пјҢйӮЈе°ұзӣ‘и§Ҷе“ӘдёӘеұһжҖ§гҖӮ

  • watchEffect жңүзӮ№еғҸ computedпјҡ

    • дҪҶ computed жіЁйҮҚзҡ„и®Ўз®—еҮәжқҘзҡ„еҖјпјҲеӣһи°ғеҮҪж•°зҡ„иҝ”еӣһеҖјпјүпјҢжүҖд»Ҙеҝ…йЎ»иҰҒеҶҷиҝ”еӣһеҖјгҖӮ
    • иҖҢ watchEffect жӣҙжіЁйҮҚзҡ„жҳҜиҝҮзЁӢпјҲеӣһи°ғеҮҪж•°зҡ„еҮҪж•°дҪ“пјүпјҢжүҖд»ҘдёҚз”ЁеҶҷиҝ”еӣһеҖјгҖӮ

з”ҹе‘Ҫе‘Ёжңҹ

  • Vue3.0 дёӯеҸҜд»Ҙз»§з»ӯдҪҝз”Ё Vue2.x дёӯзҡ„з”ҹе‘Ҫе‘Ёжңҹй’©еӯҗпјҢдҪҶжңүжңүдёӨдёӘиў«жӣҙеҗҚпјҡ

    • beforeDestroyж”№еҗҚдёә beforeUnmount
    • destroyedж”№еҗҚдёә unmounted
    • дҪҶжҳҜеҶҷеңЁеӨ–йқўи®ҝй—® setup йҮҢйқўзҡ„ж•°жҚ®дјҡеҫҲйә»зғҰ

  • Vue3.0 д№ҹжҸҗдҫӣдәҶ Composition API еҪўејҸзҡ„з”ҹе‘Ҫе‘Ёжңҹй’©еӯҗпјҢдёҺ Vue2.x дёӯй’©еӯҗеҜ№еә”е…ізі»еҰӮдёӢпјҡ

    • beforeCreate==>setup()
    • created==>setup()
    • beforeMount ==>onBeforeMount
    • mounted==>onMounted
    • beforeUpdate==>onBeforeUpdate
    • updated ==>onUpdated
    • beforeUnmount ==>onBeforeUnmount
    • unmounted ==>onUnmounted

иҮӘе®ҡд№ү hook еҮҪж•°

  • д»Җд№ҲжҳҜ hookпјҹвҖ”вҖ” жң¬иҙЁжҳҜдёҖдёӘеҮҪж•°пјҢжҠҠ setup еҮҪж•°дёӯдҪҝз”Ёзҡ„ Composition API иҝӣиЎҢдәҶе°ҒиЈ…гҖӮ
  • зұ»дјјдәҺ vue2.x дёӯзҡ„ mixinгҖӮ
  • иҮӘе®ҡд№ү hook зҡ„дјҳеҠҝ: еӨҚз”Ёд»Јз Ғ, и®© setup дёӯзҡ„йҖ»иҫ‘жӣҙжё…жҘҡжҳ“жҮӮгҖӮ

еңЁ src дёӢж–°е»ә hooks ж–Үд»¶еӨ№пјҢйҮҢйқўж”ҫиҰҒйҮҚеӨҚз”Ёзҡ„ js ж–Үд»¶пјҢз„¶еҗҺеҜјеҮәеј•е…ҘеҚіеҸҜеӨҚз”Ё

  • usePoint.js

  • import { reactive, onMounted, onBeforeUnmount } from "vue";
export default function () {
  let p = reactive({
    x: 0,
    y: 0,
  });

  function savePoint(e) {
    console.log(e.pageX, e.pageY);
    p.x = e.pageX;
    p.y = e.pageY;
  }

  onMounted(() => {
    window.addEventListener("click", savePoint);
  });

  onBeforeUnmount(() => {
    window.removeEventListener("click", savePoint);
  });
  return p;
}

е…¶д»– Composition API

1.shallowReactive дёҺ shallowRef

  • shallowReactiveпјҡеҸӘеӨ„зҗҶеҜ№иұЎжңҖеӨ–еұӮеұһжҖ§зҡ„е“Қеә”ејҸпјҲжө…е“Қеә”ејҸпјүгҖӮ
  • shallowRefпјҡеҸӘеӨ„зҗҶеҹәжң¬ж•°жҚ®зұ»еһӢзҡ„е“Қеә”ејҸ, дёҚиҝӣиЎҢеҜ№иұЎзҡ„е“Қеә”ејҸеӨ„зҗҶгҖӮ
  • д»Җд№Ҳж—¶еҖҷдҪҝз”Ё?
    • еҰӮжһңжңүдёҖдёӘеҜ№иұЎж•°жҚ®пјҢз»“жһ„жҜ”иҫғж·ұ, дҪҶеҸҳеҢ–ж—¶еҸӘжҳҜеӨ–еұӮеұһжҖ§еҸҳеҢ– ===> shallowReactiveгҖӮ
    • еҰӮжһңжңүдёҖдёӘеҜ№иұЎж•°жҚ®пјҢеҗҺз»ӯеҠҹиғҪдёҚдјҡдҝ®ж”№иҜҘеҜ№иұЎдёӯзҡ„еұһжҖ§пјҢиҖҢжҳҜз”ҹж–°зҡ„еҜ№иұЎжқҘжӣҝжҚў ===> shallowRefгҖӮ

дёӘдәәж„ҹи§үе°ұжҳҜзӣёеҪ“дәҺеҹәжң¬ж•°жҚ®

2.readonly дёҺ shallowReadonly

  • readonly: и®©дёҖдёӘе“Қеә”ејҸж•°жҚ®еҸҳдёәеҸӘиҜ»зҡ„пјҲж·ұеҸӘиҜ»пјүгҖӮ
  • shallowReadonlyпјҡи®©дёҖдёӘе“Қеә”ејҸж•°жҚ®еҸҳдёәеҸӘиҜ»зҡ„пјҲжө…еҸӘиҜ»пјүгҖӮ
  • еә”з”ЁеңәжҷҜ: дёҚеёҢжңӣж•°жҚ®иў«дҝ®ж”№ж—¶гҖӮ

3.toRaw дёҺ markRaw

  • toRawпјҡ
    • дҪңз”Ёпјҡе°ҶдёҖдёӘз”ұreactiveз”ҹжҲҗзҡ„е“Қеә”ејҸеҜ№иұЎиҪ¬дёәжҷ®йҖҡеҜ№иұЎгҖӮ
    • дҪҝз”ЁеңәжҷҜпјҡз”ЁдәҺиҜ»еҸ–е“Қеә”ејҸеҜ№иұЎеҜ№еә”зҡ„жҷ®йҖҡеҜ№иұЎпјҢеҜ№иҝҷдёӘжҷ®йҖҡеҜ№иұЎзҡ„жүҖжңүж“ҚдҪңпјҢдёҚдјҡеј•иө·йЎөйқўжӣҙж–°гҖӮ
  • markRawпјҡ
    • дҪңз”Ёпјҡж Үи®°дёҖдёӘеҜ№иұЎпјҢдҪҝе…¶ж°ёиҝңдёҚдјҡеҶҚжҲҗдёәе“Қеә”ејҸеҜ№иұЎгҖӮ
    • еә”з”ЁеңәжҷҜ:
      1. жңүдәӣеҖјдёҚеә”иў«и®ҫзҪ®дёәе“Қеә”ејҸзҡ„пјҢдҫӢеҰӮеӨҚжқӮзҡ„з¬¬дёүж–№зұ»еә“зӯүгҖӮ
      2. еҪ“жёІжҹ“е…·жңүдёҚеҸҜеҸҳж•°жҚ®жәҗзҡ„еӨ§еҲ—иЎЁж—¶пјҢи·іиҝҮе“Қеә”ејҸиҪ¬жҚўеҸҜд»ҘжҸҗй«ҳжҖ§иғҪгҖӮ
  • toRaw дёҖиҲ¬з”ЁеңЁжҠҠе·Із»Ҹз”ұ reactive з”ҹжҲҗзҡ„е“Қеә”ејҸж•°жҚ®иҪ¬жҚўеӣһжҷ®йҖҡеҜ№иұЎ
  • markRaw дҪҝз”ЁеңәжҷҜжӣҙеӨҡпјҢеӣ дёәе°Ҷжҷ®йҖҡеҜ№иұЎиҪ¬жҚўдёәе“Қеә”ејҸеҜ№иұЎйңҖиҰҒе°ҸеҸ·жҖ§иғҪпјҢиҝҷеҸҜд»ҘжҠҠдёҖдәӣеӨҚжқӮе№¶дё”дёҚйңҖиҰҒж”№еҸҳзҡ„еҜ№иұЎдёҚеҺ»иҪ¬еҢ–пјҢеҸҜд»ҘиҠӮзәҰжҖ§иғҪ

  • дҪңз”ЁпјҡеҲӣе»әдёҖдёӘиҮӘе®ҡд№үзҡ„ refпјҢе№¶еҜ№е…¶дҫқиө–йЎ№и·ҹиёӘе’Ңжӣҙж–°и§ҰеҸ‘иҝӣиЎҢжҳҫејҸжҺ§еҲ¶гҖӮ
